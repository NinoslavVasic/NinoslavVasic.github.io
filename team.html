Welcome to the Patent Foundation's open marketplace. This market serves to raise funds for the Patent development project by auctioning Patents interested parties. Using this interface, patent owners can create record ownership, licenses, and assignments. Each transaction is stored permanently on Ethereum and an Inter-Planteary File System (IPFS) called Pinata. Note you will need Pinata API Keys. Navigate to Kovan Etherscan to explore the transactions performed by the smart contract on Kovan Etherscan.
